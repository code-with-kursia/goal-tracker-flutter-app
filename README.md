# 🎯 Personal Finance & Goal Tracker App

A fully functional, data-centric mobile application built with Flutter to help users monitor financial targets and track daily income. 

## ✨ Key Features
* *Dynamic Goal Progress:* Features a clean circular progress indicator that automatically calculates and displays the goal achievement percentage.
* *Real-time Activity Feed:* A structured transaction log that updates instantly when a new income source is added.
* *User Personalization:* Fully functional settings and input screens to manage monthly targets seamlessly.
* *Pixel-Perfect UI:* High-fidelity design translated perfectly from Figma to functional Dart code.
